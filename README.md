# Natural Language to SQL Query Translator

A Python-based desktop application that converts natural language questions into SQL queries and executes them on an SQLite database. The application provides a simple graphical interface, making database querying easier for users who are not familiar with SQL.

---

## Project Overview

The main objective of this project is to simplify database interaction by allowing users to enter queries in natural language. The application translates the input into SQL, executes the query on the database, and displays the results through an easy-to-use interface.

---

## Features

- User authentication (Login)
- Natural language to SQL conversion
- SQLite database integration
- Automatic SQL query execution
- Interactive graphical user interface
- Displays query results instantly
- Easy-to-use desktop application

---

## Technologies Used

- Python
- Tkinter
- SQLite
- SQL
- Natural Language Processing (Rule-based)

## Installation

1. Clone the repository

```bash
git clone https://github.com/FathimathShaheema/nltosql_app.git
```

2. Open the project folder.

3. Install the required dependencies.

4. Run the application.

```bash
python main.py
```
